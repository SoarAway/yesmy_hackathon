# YESMY Hackathon Landing Page

React + Vite landing page prototype for a YES 5G campaign experience.

## Overview

This project is a front-end hackathon build focused on:

- a branded YES intro splash
- a responsive landing page experience (mobile + desktop)
- promo/deals browsing cards
- a dedicated `Quick Tools` area with an AI chatbot UI section

## Tech Stack

- React 18
- Vite 5
- Tailwind CSS 3

## Run Locally

Prerequisites:

- Node.js 18+
- npm

Install dependencies:

```bash
npm install
```

Start dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

## Key UI Sections

- **Intro Splash**: full-screen promo banner shown for a few seconds before entering the page
- **Hero Banner**: reduced-height first section so users can immediately see more content
- **Deals Hub**: horizontal card carousel for featured offers
- **Quick Tools / AI Assistant**: intuitive chatbot-style block with quick prompts and input area
